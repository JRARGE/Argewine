Dos formas de cargar los costos de FedEx.
La más confiable es exportar el reporte desde FedEx Billing Online (CSV/Excel) — trae envíos y duties de todas las facturas juntos, sin depender de leer PDFs. También podés subir los PDF individuales para que los lea la IA.
Importar reporte de FedEx Billing Online (recomendado)

En FedEx Billing Online, exportá el reporte de facturas a CSV o Excel. Buscá el que tenga las columnas Invoice Type, Shipper Reference y Air Waybill Total Amount. Subilo acá y se cargan todos los envíos y duties de una, cruzados por número de pedido.

Google Drive — almacenamiento persistente

Los datos se guardan en un archivo argewine_data.json en tu Google Drive. Funciona desde cualquier dispositivo. Necesitás autorizar el acceso una sola vez.

Sin Google Drive conectado. Los datos se guardan solo en este navegador (localStorage). Conectá Drive para persistencia total.
Setup inicial (una sola vez):
1. Creá un OAuth 2.0 Client ID en console.cloud.google.com
  → APIs & Services → Credentials → Create Credentials → OAuth Client ID
  → Tipo: Web application
  → En "Authorized JavaScript origins" agregá la URL desde donde usás la app
  → En "Authorized redirect URIs" agregá la misma URL
2. Pegá el Client ID abajo y guardá
3. Tocá "Conectar con Google Drive" → se abre un popup de Google para autorizar
4. Los datos se sincronizan automáticamente en cada guardado
